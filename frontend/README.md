# Frontend

Frontend source lives in the existing `client/` folder.

Main paths:
- `client/src/pages` - route pages
- `client/src/components` - reusable UI and feature components
- `client/src/hooks` - React Query and auth hooks
- `client/src/lib` - client utilities
- `client/public` - static assets

Key entry points:
- `client/src/App.tsx`
- `client/index.html`
